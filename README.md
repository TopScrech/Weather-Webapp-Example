# Weather Atlas

Apple Weather-inspired website powered by Bun + React + Vite

## Features

- Live current conditions for searched cities or your geolocation
- 36-hour forecast strip with weather icons and rain chance
- 10-day forecast with min/max temperature bars
- Key weather metrics: UV index, humidity, dew point, wind, pressure, visibility
- Sun and moon panel with daylight progress and moon phase
- Air quality panel with US AQI trend and pollutant breakdown
- Rain outlook chart for the next 12 hours
- Dynamic sky themes based on day/night and weather conditions
- Fahrenheit and Celsius switching

## Run

```bash
bun install
bun run dev
```

## Build

```bash
bun run build
bun run preview
```

## Data Providers

- Open-Meteo Forecast API
- Open-Meteo Geocoding API
- Open-Meteo Air Quality API
